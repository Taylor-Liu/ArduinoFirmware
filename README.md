# ArduinoFirmware
Motion controller firmware. Release the motor driver v1.0 firmware

## Purpose
[ArduinoFirmware](https://github.com/open-robot/ArduinoFirmware) is the Motion controller firmware of [Open Source Robot](https://github.com/open-robot). 
You can also find the [hardware](https://github.com/open-robot/Hardware) and [main ros package](https://github.com/open-robot/open-robot) repository in that project.   
Suggestion: If you want to build your own robot, our repositories may save your time.  
More information about Open Source Robot on our [web page.](http://www.ros-robot.com/)
